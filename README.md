# Roboto

## How to install Roboto ?
1. Download source files
2. Import source file to your IDE and modify admin id and creator ID
3. Compile
4. Fill bot.properties ! The telemetry is optional
5. Enjoy !

## About telemetry

The telemetry work with CachetHq, for the graph and the bot status.
To connect API and the bot to CachetHq, i use my own library cachet2bot.
